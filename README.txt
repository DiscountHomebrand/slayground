About the game:
Slayground is our first person shooter set in a post-apocalyptic town where the player must kill 100 zombies to survive.

How to load the game:
OPen project in unity
File > build and run

Main menu of game:
Press Play to start the game
Press Quit to quit the game
Press Controls to read controls

Controls:
Movement - W A S D 
Camera - Mouse
Shoot - Left Click
Jump - Space
Pause - Esc
